1.3. Ресурсы «Справочник»
Назначение
Служит для быстрого получения справочной информации студентом при решении задач, прохождении самопроверки, разборе задач практики.
Структура
Материал сгруппирован по отдельным темам, возможно два способа представления информации. Первый предполагает использование таблиц, удобен при перечислении большого количества формул по теме, второй предполагает простое перечисление основных формул, теорем, методов решения и т. д.
