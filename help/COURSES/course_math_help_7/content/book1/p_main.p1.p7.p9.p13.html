2.2.4. Поиск по УМК
Для быстрого поиска информации по курсу в УМК реализованы средства поиска по ключевым словам.
Для активации поиска по ключевым словам следует кликнуть по значку «бинокль» в верхней части блока навигации по курсу. В итоге в области просмотра откроется дополнительная вкладка «Поиск».
Структура вкладки «Поиск»
Вкладка «Поиск» состоит из линейки инструментов управления поиском и расположенной под ней области просмотра результатов поиска.
Линейка инструментов управления поиском включает:
  • Поле ввода ключевого слова
  • Выпадающий список типов логических блоков выделяемых из текста
  • Кнопку запуска поиска
Рис. 11. Линейка инструментов управления поиском
Работа с вкладкой «Поиск»
Введите в поле ввода ключевое слово или часть слова, которое вы хотите найти. Минимальное число символов для поиска установлено равным трем.
По умолчанию поиск осуществляется по всему тексту УМК. Однако, пользователь может выбрать тип логического блока, в котором будет осуществляться поиск нужной ему информации. Настройка параметров того, где следует искать, осуществляется с помощью выбора соответствующего типа блока в выпадающем списке.
При этом вы можете как в ручную вводить название блока, в котором следует проводить поиск, так и выбирать нужный тип блока из выпадающего списка.
Для запуска процедуры поиска нажмите кнопку «Искать» (кнопка с нарисованной лупой).
Если по введенному вами ключевому запросу ничего не будет найдено, в области просмотра результатов поиска отобразится соответствующее сообщение.
Рис. 12. Сообщение сигнализирующее, что ничего не найдено по введенному запросу
Если в тексте учебника найдено введенное вами ключевое слово, то в области просмотра отобразится наименования блоков, в которых найдено слово.
Для перехода к результату поиска кликните по имени блока в результатах поиска, в итоге вы перейдете в раздел, содержащий слово, которое вы искали.
Слова поискового запроса выделяются красным цветом в основном тексте УМК.
