2.3.2. Панель выбора методических типов
Панель выбора методических типов позволяет отображать на экране только тот тип логических блоков, который необходим пользователю.
Рис. 18. Панель быстрого перехода к различным типам материалов
Всего существует 9 типов логических блоков:
  • Определения
  • Примеры
  • Следствия
  • Задачи
  • Свойства
  • Доказательства
  • Упражнения
  • Теоремы
  • Примечание
  • Задачи для самостоятельной работы
Выбор интересующего типа блока осуществляется кликом по его названию. Например, на рис. 19. изображен выбор отображения только логических блоков типа «Примеры».
 
Рис. 19. Выбор показа примеров
Для отображения всего текста УМК кликните по кнопке «Все». Пользователь может задать одновременное отображение сразу нескольких типов блоков. Для этого следует нажать кнопки, соответствующие требуемым типам блоков.
