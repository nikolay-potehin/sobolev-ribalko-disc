2.1. Блочная структура интерфейса УМК «Математика»
Окно плеера электронных курсов имеет блочную структуру, и может быть разделено на три независимых взаимодополняющих блока:
  • блока навигации по курсу,
  • основной области просмотра,
  • дополнительного блока для управления ссылками, заметками и закладками.
По умолчанию открыты все три блока, по желанию пользователя блок навигации по курсу и дополнительный блок могут быть скрыты.
Размеры блоков могут также изменяться в соответствии с пожеланиями пользователя. Для того чтобы изменить размер блока, необходимо поднести курсор к линии, разделяющей блоки и когда обычный курсор мыши сменится на стрелочки направленные в разные стороны, следует кликнуть по линии и перенести её влево или вправо (Рис. 1)

Приведем краткую характеристику каждого из блоков:
  • Блок навигации по курсу: содержит дерево курса, средства для фильтрации этого дерева по методическому типу ресурса.
  • Основная область просмотра: предназначена для отображения наполнения курса, имеет систему вкладок для возможности одновременной работы с разными разделами курса.
  • Дополнительный блок: предоставляет средства для работы со ссылками заметками и закладками.
