2.2.3. Фильтрация материалов УМК по типу ресурса
В виду того, что иногда требуется получить быстрый доступ к ресурсам определенного типа, в блок навигации по курсу введен фильтр по типу ресурсов.
Фильтр расположен в верхнем левом углу блока навигации по курсу, и реализован в виде выдающего списка, в котором перечислены все возможные типы ресурсов, представленных в УМК.
По умолчанию фильтр выключен, пользователь может просматривать все дерево курса целиком.
Для активации фильтра можно:
  • Кликнуть в поле с надписью «Выберите фильтр» и ввести первые буквы типа интересующего ресурса, например лекция (Рис. 10а.)
  • Выбрать необходимый тип материала из списка, кликнув на соответствующую кнопку. (Рис. 10б.)
Рис. 10. Выбор фильтра
Для выключения фильтра выберите из выпадающего списка пункт «Нет»
