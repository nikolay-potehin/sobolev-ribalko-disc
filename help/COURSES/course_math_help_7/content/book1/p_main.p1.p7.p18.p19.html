2.4.1. Работа с заметками
Назначение
Заметки предназначены для создания пользователем комментариев к отдельным блокам текста. Заметки ставятся на уровне абзаца или целого логического блока.
Создание заметок
Для создания заметки наведите курсор мыши на интересующий вас блок. Он должен выделиться пунктирной линией. В правом верхнем углу блока выберите иконку указанную на рисунке. В итоге откроется окно создания заметки.
Рис. 24. Иконка создания заметки
Рассмотрим окно «Создание заметки».
Диалоговое окно создания закладок состоит из области ввода и линейки инструментов форматирования текста.
Рис. 25. Общий вид окна «Создание заметки»
Инструменты форматирования текста предоставляют базовые средства для форматирования текста заметок и хорошо знакомы любому пользователю, имевшему дело с текстовыми редакторами.
Перечислим эти инструменты:
  1. Выпадающий список с перечнем шрифтов
  2. Включение полужирного начертания
  3. Включение курсивного начертания
  4. Включение подчеркивания
  5. Надстрочный индекс
  6. Подстрочный индекс
  7. Выбор цвета для текста
  8. Выбор выделения определенным цветом
  9. Выравнивание по левому краю, центру и левому краю
  10. Установка ссылки
  11. Маркированный и нумерованный список
После окончания редактирования или создания заметки кликните по кнопке «OK».
Рис. 26. Инструменты форматирования текста заметки
Пользователь может получить быстрый доступ к созданным заметкам через дополнительный блок управления закладками, заметками и ссылками.
Рис. 27. Панель работы с заметками
При желании можно удалить созданные заметки нажав на кнопку «Удалить».
