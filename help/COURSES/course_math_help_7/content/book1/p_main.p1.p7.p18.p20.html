2.4.2. Работа с закладками
Назначение
Закладки предназначены для возможности быстрого перехода пользователя к определенному месту в тексте УМК, с которым он работал ранее.
Создание закладок
Для установки закладки наведите курсор мыши на блок, на который следует поставить закладку. Кликните по иконке указанной на рисунке.
Рис. 28. Иконка создания закладки
Откроется диалоговое окно ввода названия закладки. Введите название закладки и нажмите кнопку «ОК» для подтверждения, или кнопку «Отмена», если вы передумали устанавливать закладку.
Рис. 29. Вид окна ввода названия закладки
Перечень установленных закладок отображается в области «Закладки» дополнительного блока. Закладки могут быть удалены при нажатии на кнопку «Удалить».
