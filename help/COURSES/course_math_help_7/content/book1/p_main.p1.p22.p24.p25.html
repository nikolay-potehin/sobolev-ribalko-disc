3.2.1. Закрытые вопросы
Закрытые вопросы предлагают перечень ответов на поставленный вопрос, из которых пользователь должен выбрать правильный.
При этом возможны два случая:
  1. В перечне приведенных ответов только один правильный, или несколько правильных, но достаточно указать только один. Варианты ответов помечены кружочками, пользователь должен кликнуть в тот из них, который стоит напротив верного по мнению пользователя ответа.
  2. В перечне приведенных ответов несколько правильных и следует отметить все правильные варианты. Ответы отмечаются квадратиками, пользователь должен поставить галочки (кликнуть в квадратике) напротив тех ответов, которые он считает правильными. При этом если хотя бы один из вариантов ответов окажется не правильным, вопрос не засчитывается.
